This is the folder "mesh_structure" containing gromacs compatible files for Martini-3 model peptidoglycan for Escherichia coli. 
The peptidoglycan network consists of 8 chains, each 16 disaccharides long. There are total 2752 CG beads. 
Each chain has its terminal NAG bonded with its other terminal NAM at periodic boundary to mimic infinitely long chain.   
The adjacent chains are cross-linked, total number of peptide cross-linkages = 32.
